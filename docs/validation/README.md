
This folder contains validation reports for different dates. Each test date has a folder with a note summarizing and visualizing the results of each model component, and the complete system.

A template to build such reports is in [solarMED_validation_report_YYYYMMDD](YYYYMMDD/solarMED_validation_report_YYYYMMDD.md).

They are automatically built from the notebook [solarMED_validation_report.ipynb](../../solarMED_validation_report.ipynb)
